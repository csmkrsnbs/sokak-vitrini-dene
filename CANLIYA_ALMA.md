## Vercel

1. Vercel'de projeyi içe aktarın.
2. Framework: Next.js.
3. Build Command: `npm run build`.
4. Node.js: 22.x.
5. İsteğe bağlı `NEXT_PUBLIC_APP_URL` değişkenini canlı alan adıyla ekleyin.
6. Deploy edin.

RunPod, GPU, Docker, Python worker veya VTON ortam değişkeni gerekmiyor.
